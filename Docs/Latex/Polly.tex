\section{Observáveis}

Numa poligonal existirão estações fixas, vértices de coordenadas conhecidas, e estaçoes não fixas, vértices de coordenadas que se pretende determinar. Através do estacionamento nos diversos vértices da poligonal retirar-se-ão observações visando a estação atrás e à frente tanto na directa como na inversa, obtendo-se assim 4 visadas em cada estacionamento. As observáveis são as seguintes:

	\underline{Estação}:
	\begin{itemize}
	\item ID
	\item Altura Estação
	\end{itemize}

	\underline{Vizadas}:
	\begin{itemize}
	\item ID
	\item Medição Angular
	\item Distância Inclinada
	\item Altura Alvo
	\end{itemize}

\section{Modelos Matematicos}
	
	Coordenadas:
	\begin{equation*}
	E_j = \sin (R_{ij}) * DH_{ij} + E_i + \epsilon
	\end{equation*}	
	
	\begin{equation*}
	N_j = \cos (R_{ij}) * DH_{ij} + N_i + \epsilon
	\end{equation*}	

	\begin{equation*}
	Z_j = \cos (\measuredangle z_{ij}) * DI_{ij} + Z_i - (h_j - h_i) + \epsilon
	\end{equation*}	
	
	Rumos:
	\begin{equation*}
	R_{ij} = R_{0} + \measuredangle _j + \epsilon (R_0)
	\end{equation*}		

	\begin{equation*}
	R_{0} = R_{-1} + 180 \degree - \measuredangle _{i-1} + \epsilon (R_0)
	\end{equation*}		
	
	\subsection{Parametros Desconhecidos}
	\begin{itemize}
	\item Coordenadas E, N e Z de cada estação 	
	\item Rumos do limbo de cada estacionamento
	\end{itemize}

	Sendo n o número de vértices desconhecidos da poligonal, e m o número de estacionamentos, o número de parâmetros desconhecidos será: 3n + m
